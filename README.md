# ble-server
## Requirements:
1.  ### Operating System:
    * Ubuntu Mate Xenial 16.04.3 LTS for Raspberry Pi (RPI 2, 3).
    * Other Linux distributions meeting requirements.  
2.  ### Unix packages:
    * bluez >= 5.41
    * python >= 3.5.2
3.  ### Python modules:
     * bluepy >= 1.1.4
     * ponyorm >= 0.7.3
